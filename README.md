# Gestor de Productos

Aplicación web para gestionar productos que permite agregar, eliminar y visualizar productos con persistencia de datos en el navegador.

🔗 Demo: https://nahueldaniel.github.io/gestor-productos/

## Tecnologías utilizadas

- HTML
- CSS
- JavaScript
- LocalStorage

## Funcionalidades

- Agregar productos
- Eliminar productos
- Visualizar lista de productos
- Persistencia de datos (no se borran al recargar)

## Aprendizajes

- Manejo del DOM
- Uso de eventos en JavaScript
- Persistencia de datos con LocalStorage
- Manipulación de arrays y objetos
